# shoppers_bay
A ecommerce website using php, Laravel, Html, css, Ajax, Jquery
Key Features:<br>
User:<br>
->can register<br>
->can sign in<br>
->can add products to cart<br>
->can checkout<br>
->can make payment through Bkash/Rocket transaction Id.<br>
<br>
Admin:<br>
->can login<br>
->can review orders<br>
->can approve payments<br>
